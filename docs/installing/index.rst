============================
Installing the LOCKSS System
============================

.. note::

   .. COMMENT LATESTVERSION

   .. COMMENT PREVIOUSVERSION

   This chapter describes how to install the LOCKSS 2.0-beta1 system from scratch. If you are upgrading an existing LOCKSS 2.0-alpha6 system to 2.0-beta1, please see the upgrade instructions in the previous chapter:

   .. button-ref:: /upgrading/index
      :ref-type: doc
      :color: primary
      :align: center

.. tip::

   Before you begin installing the LOCKSS system, we strongly recommend you first bring your operating system up to date by applying security updates and upgrading installed packages. Ask your system administrator or see :doc:`/sysadmin/os-updates`.
