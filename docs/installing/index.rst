============================
Installing the LOCKSS System
============================

.. note::

   This chapter describes how to install the LOCKSS |LATEST_MINOR| system from scratch. If you are upgrading an existing LOCKSS 2.0-alpha5, 2.0-alpha6 or 2.0.71-alpha7 system to |LATEST_PATCH|, please see the upgrade instructions in the previous chapter:

   .. button-ref:: /upgrading/index
      :ref-type: doc
      :color: primary
      :outline:
      :align: center

.. tip::

   Before you begin installing the LOCKSS system, we strongly recommend you first bring your operating system up to date by applying security updates and upgrading installed packages. Ask your system administrator or see :doc:`/sysadmin/os-updates`.
