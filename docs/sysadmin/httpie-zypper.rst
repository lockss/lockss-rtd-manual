To install HTTPie with Zypper, run these commands (as ``root``):

1. Run this :program:`zypper` command:

   .. code-block:: shell

      zypper refresh

2. Run this :program:`zypper` command:

   .. code-block:: shell

      zypper --non-interactive install httpie

   or equivalently:

   .. code-block:: shell

      zypper -n install httpie
