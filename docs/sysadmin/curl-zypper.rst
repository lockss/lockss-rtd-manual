To install Curl with Zypper, follow these steps (as ``root``):

1. Run this :program:`zypper` command:

   .. code-block:: shell

      zypper refresh

2. Run this :program:`zypper` command:

   .. code-block:: shell

      zypper --non-interactive install curl

   or equivalently:

   .. code-block:: shell

      zypper -n install curl
